Introduction to Python

Problem Statement
Understand how Python code is executed and print a welcome message to the console.
Explanation

Python is an interpreted, dynamically-typed language, which means there's no separate compile step — the interpreter reads and executes your code line by line.

Unlike C++, Python does not require explicit type declarations or a `main()` entry point — the file itself runs top to bottom when executed. Indentation defines code blocks.
