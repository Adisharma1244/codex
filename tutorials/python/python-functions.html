Functions & Recursion

Problem Statement
Write a function that computes the factorial of a number using recursion.
Explanation

A function is defined with `def`. Recursion is when a function calls itself to solve a smaller version of the same problem. Every recursive function needs a base case and a recursive case.

Code
Python
def factorial_recursive(n): if n == 0: return 1 return n * factorial_recursive(n - 1) print("Recursive 5! =", factorial_recursive(5))
Recursive 5! = 120
Complexity Analysis
Time Complexity
O(n)
Space Complexity
O(n)
