Loops: for and while

Problem Statement
Print numbers 1 through 5 using both a for loop and a while loop.
Explanation

Python has two loop constructs: `for` loop (iterates directly over a sequence like `range()`) and `while` loop (repeats as long as a condition is true).

Code
Python
print("for loop:", end=" ") for i in range(1, 6): print(i, end=" ") print() print("while loop:", end=" ") j = 1 while j <= 5: print(j, end=" ") j += 1 print()
for loop: 1 2 3 4 5 while loop: 1 2 3 4 5
Complexity Analysis
Time Complexity
O(n)
Space Complexity
O(1)
