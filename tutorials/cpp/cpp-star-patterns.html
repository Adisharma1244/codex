Nested Loops & Star Patterns

Problem Statement
Print a right-angled star pyramid of height n, where row i contains i stars.
Explanation

Star pattern problems are the classic way to build intuition for nested loops. The outer loop controls the row number (`i`), and the inner loop controls how many stars are printed in that row.

The key insight: for row `i`, the inner loop runs from `j = 1` to `j = i`, printing one star per iteration, then a newline ends the row.

Code
C++
#include <iostream> using namespace std; int main() { int n = 5; for (int i = 1; i <= n; i++) { for (int j = 1; j <= i; j++) { cout << "* "; } cout << endl; } return 0; }
* * * * * * * * * * * * * * *
Complexity Analysis
Time Complexity
O(n^2)
Space Complexity
O(1)
